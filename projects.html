Projects

Here are two small snippets of projects I have worked on over the past couple of years when I have found free time between school, work, and family. Given I was working on my advanced degree from Fall 2022 until now, my daughter was born in Fall 2022, and I've been fully employed this whole time, I haven't had too much free time to devote to passion projects. That said, I've kept myself busy and have found some time to build a couple of projects that I'm proud of!

Stock Market Lightsaber

As a fan of Star Wars with an interest in the current market performance of various stocks, I created a visual indicator of the current performance of my stock portfolio. Using the free version of the Python API from Finnhub, I wrote a program to continuously look up the value of a portfolio (stored as a JSON file). Based on whether the current portfolio value is greater than the portfolio value at close of the previous day, I determine whether my day value has increased (green) or decreased (red). If I'm having internet connectivity problems, or if there is a different error, I set the color to blue to indicate no current results.
Given the Star Wars universe has many additional colors, I created additional functions, operable by a button on the hilt, to cycle through different colors and modes. One other functional mode sets the color on a gradient from blue to red based on the current temperature. All of the code is written in Python and the initial version is saved here. The lightsaber hilt is 3D printed with a custom design based on Yoda's handle and is powered with a 5V DC plug wired directly to a Raspberry Pi Zero and the individually addressible RGB LED strip. It was quite a fun project that I undertook in the winter break of my Master's program in December 2023.

Metro Board

After my brother sent me a screenshot of a Reddit Post he found, I decided to make him a Metro Board for Christmas 2022. In late February 2023, I delivered on my goal. Taking the main source code from the original creater, I built some minor additions to wish family members Happy Birthday or celebrate special national holidays like April Fools (turning the sign in to a Hot Dog Stand) or National Lawyer's Day.
Overall this was a great project to learn a bit about controlling a LED matrix sign. Additionally, the established code base helped with the baseline functionality whereas I was able to focus more on making it special and unique with the packaging, aesthetics, and additional Easter eggs added in.
